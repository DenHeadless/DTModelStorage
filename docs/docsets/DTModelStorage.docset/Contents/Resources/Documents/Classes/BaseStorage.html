  • performUpdates(_:)

    Performs update block in storage. After update is finished, delegate will be notified. Parameter block: Block to execute

    Note

    This method allows to execute several updates in a single batch. It is similar to UICollectionView method performBatchUpdates:.

    Warning

    Performing mutually exclusive updates inside block can cause application crash.
